Exhibitionist is a heighly features image gallery plugin for wordpress.

== Description ==

Exhibitionist Features:

	Image Transitions:
		*Crossafe
		*Fade In->Out->In
		*None
	
	Navigation Modes:
		*Grid Thumbnails
		*Inline Thumbnails
		*Caption (Images caption has back forward buttons)
		*Automated Slideshow
	
	Image Enquiry:
		Image enquire allows users to send an email to a specified email address.
		The emails has information about the page and image the user was viewing

== Installation ==

1. Extract exhibitionist.zip to get a folder called exhibitionist/
2. Upload the exhibitionist/ folder to the `/wp-content/plugins/` directory
3. Activate Exhibitionist through the 'Plugins' menu in WordPress
4. Revise settings in the 'Settings' -> 'Exhibitionist' menu in wordpress.
4. Place [Exhibitionist] in your page or post where you would like the gallery to be displayed.
